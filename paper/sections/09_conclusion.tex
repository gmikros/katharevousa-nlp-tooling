\section{Conclusion}

We presented a reproducible Katharevousa NLP pipeline from OCR-aware reconstruction to parser benchmarking and release selection. The final frozen gold dataset contains 1,697 sentences and supports controlled comparisons across feature-based, transformer, mBERT, and Stanza baselines. The best model for first public release is XLM-R \texttt{v1\_opt}, which provides the strongest overall syntactic attachment performance on the fixed test split.

Future work should prioritize (a) targeted expansion of high-quality gold data, (b) confidence-calibrated error analysis for long-distance dependencies, and (c) external validation on out-of-domain Katharevousa genres.
