# Paper Folder (Overleaf-ready)

This folder is structured for direct Overleaf import from GitHub.

## Files

- `main.tex`: manuscript source
- `references.bib`: bibliography
- `figures/`: place figure PDFs/PNGs here

## Overleaf

1. In Overleaf: `New Project` -> `Import from GitHub`
2. Select this repository
3. Set `paper/main.tex` as the main file if needed
